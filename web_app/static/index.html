Сессии анализа

Сессий пока нет. Создайте первую — она станет вашим комплектом документов на проверку.
Сессии общие для всех: любой сотрудник видит чужие, может открыть отчёт и снять сессию с очереди. Скриптовая часть анализа запускается сразу и считается параллельно; в очередь сессия встаёт только перед стадией нейросетей — сервер ИИ один на всех.
