Сессии анализа

Сессий пока нет. Создайте первую — она станет вашим комплектом документов на проверку.
Здесь ваши сессии: их видите только вы (администратор — все, с группировкой по пользователям). Скриптовая часть анализа запускается сразу и считается параллельно; в очередь сессия встаёт только перед стадией нейросетей — сервер ИИ один на всех.

Документы сессии

Перетащите файлы сюда
или нажмите, чтобы выбрать. PDF - схемы и чертежи, XLSX - спецификация.
    Файлы ещё не загружены.
    Все документы сессии считаются одним проектом и сверяются между собой. Состав может быть любым: схема, сборочный чертёж, спецификация, нетлист — или только часть из них.
    Модели нейросетей для этой сессии
    Выбор только из моделей, ЗАГРУЖЕННЫХ в память сервера ИИ. Пустой выбор — модель по умолчанию. Настройка действует только на эту сессию, поэтому потом всегда видно, чем считался отчёт. Полная настройка нейросетей и загрузка моделей — у администратора.
    Вкладку можно закрыть — результат дождётся вас в списке сессий.

      Чат с нейросетью

      Диалог обращается к LM Studio напрямую, минуя очередь анализа. Доступны только модели, ЗАГРУЖЕННЫЕ в память сервера. Файлы и фото передаются модели: картинки — как изображения (нужна модель со «зрением»), прочие файлы — извлечённым из них текстом. «Новый чат» закрывает текущий (он уходит в архив на диске).
      Настройки ИИ
      Управление сервером LM Studio и глобальным конфигом нейросетей. Действует на всю программу; правки конфига сохраняются в файл установки и вступают в силу со следующего прогона.
      Параметры загрузки модели
      Пустые поля — как решит сервер. Параметры применяются к следующей загрузке. GPU-offload через API LM Studio не задаётся — он берётся из настроек модели в самом LM Studio.
      Загрузка списка…
      Изменённые здесь значения переопределяют значения по умолчанию и вступают в силу со следующего запуска анализа.
      Пользователи
      Обычному пользователю нужен только логин. Администратор видит все сессии и управляет пользователями — ему нужен пароль.
